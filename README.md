# Enterprise IT Service Desk & IAM Simulation

## Overview
This project simulates a real-world IT Service Desk environment, handling user incidents, access management, and system troubleshooting aligned with enterprise workflows.

## Key Features
- 200+ IT support tickets (Login, Access, Software, Hardware issues)
- User lifecycle management (creation, modification, disablement)
- MFA & authentication issue simulation
- SLA tracking and First Call Resolution (FCR)
- Escalation handling (L1 → L2 support)
- Knowledge base for common IT issues

## Tools & Concepts
- ServiceNow (ITSM Workflow Simulation)
- Active Directory & Azure AD (User Management)
- Okta (MFA & Authentication)
- Microsoft Intune (Device Compliance Concepts)
- Excel (Ticketing & Dashboard Simulation)

## Skills Demonstrated
- Incident Management
- IT Support Troubleshooting
- SLA Handling
- Customer Support Workflow
- Documentation & Knowledge Base Creation

## Outcome
- Achieved ~85% SLA compliance
- Achieved ~80% First Call Resolution (FCR)
- Improved issue resolution efficiency through structured troubleshooting
